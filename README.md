# corotTruss example

OpenSeesPy example: **inclined 2D corotational truss** with geometric nonlinearity and load/displacement control, intended for teaching.

## Run in Colab

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/gaaraujo/corotTruss-example/blob/main/inclined_corot_truss.ipynb)

Click the badge, wait for Colab to open, then use **Runtime → Run all** in `inclined_corot_truss.ipynb`.

## Run locally (optional)

1. Clone the repo and enter the folder:
   ```bash
   git clone https://github.com/gaaraujo/corotTruss-example.git
   cd corotTruss-example
   ```
2. Install dependencies (in a virtual environment if you like):
   ```bash
   pip install -r requirements.txt
   pip install -e .
   ```
3. Open `inclined_corot_truss.ipynb` in Jupyter / VS Code and run all cells.
